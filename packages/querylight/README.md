# @tryformation/querylight-ts

Lightweight in-memory search for TypeScript applications.

`@tryformation/querylight-ts` combines structured indexing, lexical ranking, boolean queries, aggregations, vector search, geo search, and offset-based highlighting in one small API for browser and Node.js projects.

## Install

```bash
npm install @tryformation/querylight-ts
```

## Demo

- Cloudflare demo: [https://querylight-ts-demo.pages.dev](https://querylight-ts-demo.pages.dev)

The demo site also includes the most comprehensive documentation set for the project and showcases the "Ask the Docs" vector-search experience over that documentation corpus.
